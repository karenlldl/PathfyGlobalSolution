# 💜 Pathfy – Plataforma Corporativa de Upskilling e Reskilling  

##Link Vercel: https://pathfyplatform.vercel.app/
### Global Solution – FIAP (1TDSPA)

O **Pathfy** é uma plataforma corporativa projetada para ajudar empresas a desenvolverem seus colaboradores por meio de **trilhas inteligentes de aprendizado**, **análise de risco de automação** e **simulação de mobilidade interna**.  
O sistema faz uso de IA conversacional (WatsonX) para orientar o colaborador e sugerir caminhos de carreira baseados em dados corporativos.

# 🎯 Objetivo do Projeto

- Avaliar o **perfil profissional** do colaborador (área, cargo, interesses).  
- Identificar o **risco de substituição por IA** em sua função.  
- Simular uma **consulta ao banco de dados da empresa**, incluindo vagas internas e competências necessárias.  
- Recomendar **trilhas de upskilling e reskilling** alinhadas ao futuro do trabalho.  
- Ajudar empresas a **reduzirem custos**, **reter talentos** e **otimizarem treinamentos**.

---

# ❗ Problema Corporativo Resolvido

Empresas enfrentam desafios importantes em um mercado impactado pela automação:

- Cargos sendo substituídos pela IA.  
- Treinamentos genéricos que não trazem resultado.  
- Falta de clareza sobre quais competências cada colaborador deve desenvolver.  
- Baixa mobilidade interna, mesmo com vagas disponíveis.  
- Alto turnover causado por falta de perspectiva de carreira.  

O **Pathfy B2B resolve isso** fornecendo:

- Trilhas personalizadas  
- Direcionamento inteligente  
- Simulação de banco corporativo  
- Análise de risco profissional  
- Preparação do colaborador para o futuro da IA

A solução é composta por:

### ✔ Interface completa em HTML, CSS e JavaScript  
- Responsiva para desktop, tablet e mobile.  
- Header com menu hamburguer funcional.  
- Interações, validações e simulações via JavaScript.

### ✔ Chatbot (WatsonX Assistant)  
- Coleta área, cargo, risco de IA e interesses.  
- Usa intenções, entidades e slots para personalizar trilhas.  
- Retorna recomendações simuladas da Pathfy.

### ✔ Simulação de Dados Corporativos  
O sistema representa como seria a integração real com:

- Vagas internas  
- Skills corporativas  
- Mapeamento de cargos em risco  
- Trilha recomendada via IA  

### ✔ Lógica do Sistema  
1. O colaborador informa área e cargo.  
2. O sistema simula a verificação de risco profissional.  
3. O chatbot pergunta interesses.  
4. O Pathfy gera uma trilha de aprendizagem.  
5. Exibe resultados e caminhos de desenvolvimento interno.
